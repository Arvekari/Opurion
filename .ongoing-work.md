## Autonomous Execution Rules

Agent may continue automatically without confirmation when:

- Task is a UI migration
- Task is refactor only
- No database schema changes
- No API contract changes
- No destructive file deletions

If these conditions are met:
→ Do NOT ask for confirmation
→ Continue execution until task acceptance criteria are met.

Human confirmation is required only for:

- at moment there is no situation where you should need human intervention
